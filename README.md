## Hi there 👋. I build tools and find clarity in chaos

My professional life is based upon my insatiable desire to learn and understand all things. I design and implement bespoke tools that surface gaps in insight by combining disparate systems in various ways, revealing areas of potential failure and converting large volumes of information into actionable insight. Over the course of my 8-year technical career — with 3 + years spent exclusively developing .NET — I have developed a drive to tie together the technical requirements and client needs of organizations to make them run more efficiently.

**Recent Technical Developments:**

- **Hybrid Image Cache:** Developed a GUID-based sync method to move BLOB data from Azure SQL to local application server. The result was a reduction in the amount of time it took to synchronize data and a corresponding decrease in both cloud transfer costs and network overhead, resulting in faster image loads for end users.
- **System Validation:** Developed a database seeding engine using complex relational logic to create simulations of user lifecycles that could be recreated chronologically. This would allow for the recreation of months of historical use within a few hours, which provided us a realistic and reliable way to validate our applications under real-world, dependency driven scenarios.
